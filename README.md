# Honolulu ArtShow

-- working in progress --
An iOS App using Swift3 and Google Map API that will help to find the artistic hotspot in Honolulu, Hawaii, U.S.A..

## Technologies

- Swift3
- Xcode8
- iSO10
- Google Map API and Direction API
- MapKit
